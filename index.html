vegas-files

Styling the standard file selection on a form

  • home page
  • github
  • npmjs

Install

npm i vgfiles

or download here

Include files

CSS. Main file vgnav.css, default theme vgnav-theme.css

<link href="..path../dist/css/vgnav.css" rel="stylesheet" type="text/css">

JavaScript. Main file vgnav.js, or minified vgnav.min.js

<script src="..path../dist/js/vgnav.min.js"></script>

Usage

HTML markup

Markup:

<nav class="vg-nav">
    <ul class="vg-nav-main-container">
        <li class="active"><a href="/">Home page</a></li>
		<li class="dropdown">
            <a href="#">Left dropdown</a>
            <ul class="left">
                ...
            </ul>
        </li>
        <li class="dropdown-mega">
            <a href="#">Mega Menu</a>
            <div class="dropdown-mega-container">
                ...
            </div>
        </li>
        <li class="dropdown">
            <a href="#">Right dropdown</a>
            <ul class="right">
		       ...
            </ul>
        </li>
    </ul>
</nav>

Initialize

new VGNav({
  expand: 'md',
}).toggle()
