vegas-files

Styling the standard file selection on a form. Uses the JQuery library

    Install

    npm i vgfiles

    or download here

    Include files

    CSS. Main file vgfiles.css

    <link href="..path../dist/css/vgfiles.css" rel="stylesheet" type="text/css">

    JavaScript. Main file vgfiles.js, or minified vgfiles.min.js

    <script src="..path../dist/js/vgfiles.min.js"></script>

    Usage

    HTML markup

    Markup:

    
<div class="vg-files">
	<label for="vg-files-input">
		<span>
			<span class="vg-files-input__icon">+</span>
			<span class="vg-files-input__title">Choice Files</span>
		</span>
	</label>
	<div class="vg-files__info">
		<div class="vg-files__info--wrapper">
			<div>Files uploaded: <span class="vg-files__info--count"></span></div>
			<div><a href="#" class="vg-files__dismiss" data-dismiss="vg-files">Reset</a></div>
		</div>
		<ul class="vg-files__info--name"></ul>
	</div>

	<input type="file" name="files[]" id="vg-files-input" data-toggle="vg-files" multiple>
</div>
