Install

Include files

CSS. Main file vgfiles.css

<link href="..path../dist/css/vgfiles.css" rel="stylesheet" type="text/css">

JavaScript. Main file vgfiles.js, or minified vgfiles.min.js

<script src="..path../dist/js/vgfiles.min.js"></script>
